# 💻 Proyectos Destacados

## 🦝 Mapachito - Chatbot Inteligente para GreenHouse

En GreenHouse se encontraban con que muchas personas hacían preguntas muy específicas sobre productos naturales y suplementos, por ejemplo:

> “¿Para qué sirve el Omega 3?”
> “¿Qué diferencia hay entre estas harinas?”
> “¿Cuál me conviene más?”

Muchas veces esas consultas requerían conocimientos específicos o tomaban demasiado tiempo en responder.

Por eso desarrollé “Mapachito”, un chatbot inteligente conectado con IA para automatizar respuestas, mejorar la atención al cliente y facilitar la búsqueda de productos.

### Tecnologías utilizadas
- JavaScript
- Node.js
- APIs
- JSON
- DOM y Eventos
- LocalStorage

### Funcionalidades
- Búsqueda inteligente de productos
- Filtrado por categorías
- Reconocimiento de múltiples productos
- Simulación de respuestas automáticas
- Integración con IA

🔗 Proyecto:
https://github.com/francosantillan12/greenhouse
